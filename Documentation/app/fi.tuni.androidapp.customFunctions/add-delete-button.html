addDeleteButton

fun addDeleteButton(id: Int, layout: LinearLayout, buttonsBlock: LinearLayout, activity: Activity, context: Context, userBlock: LinearLayout)

Adds a Delete-button for a user

The function adds an ImageButton to a single user and sets an onclick listener to delete the whole user from the linearLayout, should it be pressed.

Parameters

id

ID of the user to be deleted

layout

The entire linearlayout full of users.

buttonsBlock

Update and Delete buttons are in a separate block.

activity

Required for runOnUiThread.

context

Context required for creating new items.

userBlock

The "block" of a single user.
